============
 PlateRebar
============

.. function:: nDMaterial('PlateRebar', matTag, newmatTag, matTag, sita)
   :noindex:

   This command is used to create the multi-dimensional reinforcement material.

   ================================   ===========================================================================
   ``matTag`` |int|                   integer tag identifying material
   ``newmatTag`` |int|                new integer tag identifying material deriving from pre-defined
                                      uniaxial steel material
   ``matTag`` |int|                   integer tag identifying uniaxial steel material
   ``sita`` |float|                   define the angle of steel layer,
                                      90 (longitudinal steel), 0 (tranverse steel)
   ================================   ===========================================================================
